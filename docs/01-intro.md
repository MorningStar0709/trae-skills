# Trae AI Agent Enhancements

This project is a **Trae AI Agent rules and skills collection**. Run `npx trae-agent-enhancements` to install globally — your Agent immediately gains rule constraints, professional skills, and persistent memory capabilities.

## What's Included

| Layer      | Count | One-Line Summary                                                                  |
| :--------- | :---- | :-------------------------------------------------------------------------------- |
| **Rules**  | 8     | Tell the Agent what it can do, what it can't, and which path to take              |
| **Skills** | 34    | Professional toolboxes covering design → coding → debugging → commit → completion |
| **Memory** | —     | Cross-session experience accumulation, smarter over time                          |

## One-Line Design Philosophy

**Rules make decisions, Skills perform actions, Trae native capabilities as the foundation (don't break or replace), original mechanisms add value (T-Shirt sizing, question thresholds), Chinese adaptation enhances experience.**

## Quick Start

```bash
npx trae-agent-enhancements
```

Follow the prompt to select your Trae edition (China or International). Installed globally for all projects.

Try these commands: `"帮我排查这个报错"` / `"先写计划再实现"` / `"提交代码"` / `"记住这个处理方式"` / `"改好了没？验证一下"`
