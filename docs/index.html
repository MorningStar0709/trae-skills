🚀 Trae Skills 工具集

专为 Windows 平台 Trae IDE 设计的 Skills 工具集，支持中英双语文档。

English | 中文

✨ 核心特性

🎯 Windows 优先设计

所有 Skills 均经过严格的 Windows 环境测试和优化，使用 PowerShell 命令。

🌐 中文用户适配

原生支持简体中文，平衡英文技术锚点，方便国际用户理解。

⚡ 稳定执行保障

完整的验证脚本确保智能体行为可靠和结果一致。

🔒 生产级质量

内置稳定性审查工具和验证工作流，企业级质量保证。

📦 包含的 Skills

agent-blueprint-architect 创建和优化 Trae Agent 配置，包含清晰的边界和触发条件。
creating-trae-rules 定义和组织 Trae 项目规则，支持多种激活模式。
skill-creator 脚手架搭建、审查和迭代新的 Trae 技能。
skill-stability-review 审计 Skills 的 Windows/Trae 兼容性和执行稳定性。

🚀 快速开始

克隆仓库并导入 Skills 到你的 Trae IDE：

git clone https://github.com/MorningStar0709/trae-skills.git
cd trae-skills

# 验证单个 Skill
python skills/skill-creator/scripts/quick_validate.py skills/<skill-name>

# 扫描 Windows 兼容性
python skills/skill-stability-review/scripts/review_skills.py --skill skills/<skill-name>
