# Trae AI Agent Enhancements

本项目是一套为 Trae IDE 打造的 **AI Agent 规则与技能集**。通过 `npx trae-agent-enhancements` 一键安装到全局，Agent 立即获得规则约束、专业技能和持续记忆能力。

## 有什么

| 层              | 数量   | 一句话职责                     |
| :------------- | :--- | :------------------------ |
| **Rules（规则）**  | 8 条  | 告诉 Agent 什么能做、什么不能做、该走哪条路 |
| **Skills（技能）** | 34 个 | 专业工具箱，覆盖设计→编码→调试→提交→收尾全流程 |
| **Memory（记忆）** | —    | 跨会话沉淀经验，越用越聪明             |

## 一句话设计哲学

**Rule 做决策、Skill 做操作、Trae 原生做底座（不破坏不替代）、原创机制做增量（T-Shirt 分档、提问阈值）、中文适配做体验。**

## 快速开始

```bash
npx trae-agent-enhancements
```

根据提示选择你的 Trae 版本（国内版或国际版），即可全局生效，所有项目可用。

试试这些指令：`"帮我排查这个报错"` / `"先写计划再实现"` / `"提交代码"` / `"记住这个处理方式"` / `"改好了没？验证一下"`
