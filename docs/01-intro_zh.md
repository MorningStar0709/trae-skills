# Trae AI Agent System

本项目是为 Trae IDE 打造的 **AI Agent 操作系统**。把 `.trae/` 目录放进项目根目录，Agent 立即获得规则约束、专业技能和持续记忆能力。

## 有什么

| 层              | 数量   | 一句话职责                     |
| :------------- | :--- | :------------------------ |
| **Rules（规则）**  | 8 条  | 告诉 Agent 什么能做、什么不能做、该走哪条路 |
| **Skills（技能）** | 33 个 | 专业工具箱，覆盖设计→编码→调试→提交→收尾全流程 |
| **Memory（记忆）** | —    | 跨会话沉淀经验，越用越聪明             |

## 一句话设计哲学

**Rule 做决策、Skill 做操作、Trae 原生做底座（不破坏不替代）、原创机制做增量（T-Shirt 分档、提问阈值）、中文适配做体验。**

## 快速开始

将本项目的 `.trae/` 目录复制到你的 Trae 项目根目录，即可生效。

试试这些指令：`"帮我排查这个报错"` / `"先写计划再实现"` / `"提交代码"` / `"记住这个处理方式"` / `"改好了没？验证一下"`
